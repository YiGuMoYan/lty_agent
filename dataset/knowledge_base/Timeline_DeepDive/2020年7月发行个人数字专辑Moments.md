# 2020年7月: 发行个人数字专辑《Moments》

# 洛天依个人数字专辑《Moments》发行档案

## 基本信息

- **事件名称**: 发行个人数字专辑《Moments》
- **发生时间**: 2020年7月12日中午12:00（北京时间）
- **发生地点**: 中国（线上发行）
- **相关人物/主体**: 洛天依
- **事件性质**: 音乐作品发行

## 事件概述

2020年7月12日中午12:00（北京时间），虚拟歌手洛天依在其出道八周年之际，正式发行首张官方个人EP暨首张数字专辑《Moments》。该专辑为“洛天依诞生祭”企划的核心内容，由Vsinger出品，在网易云音乐等平台上线发售。具体发布时间经Vsinger官方微博及B站动态确认。

## 备注

- 本专辑是洛天依的首张官方个人EP及首张数字专辑。
- 专辑创作体现了以用户原创（UGC）为核心的创作生态，官方宣传材料明确指出内容贡献比例达90%，实现了从社区创作到官方发行的整合闭环。
- 实体专辑的先行预售信息由Vsinger官方微博于2021年1月26日发布，为2020年度Vsinger系列发行中唯一具备实体形式的作品；正式发售日期尚未在参考资料中提供。
