# 2017年12月30日：V4声库发布

# 洛天依 V4 声库发布事件（2017年12月30日）

## 事件时间  
2017年12月30日

## 发布背景与发布方  
2017年12月30日，上海禾念信息科技有限公司正式发售《Vocaloid 4 Library: 洛天依》中文声库。该声库基于 Yamaha 的 VOCALOID™4 语音合成引擎开发，标志着洛天依从 VOCALOID 3 声库向新一代语音合成技术的全面升级。

## 声音提供者（CV - Character Voice）  
本次 V4 中文声库的录音工作由**山新**完成。作为洛天依原始声音的提供者，山新的再次献声确保了角色音色的高度一致性与专业水准。

## 技术特性  
此次发布的 V4 声库在音质、表现力和兼容性方面实现多项重要提升：

- **双音色配置**：包含“萌”与“凝”两种音色，分别代表更年轻活泼与更成熟稳重的声线，显著拓展音乐创作的表现维度。
- **新增发声参数**：
  - 支持 **Growl**（咆哮音），增强声音张力，适用于摇滚、电子等风格；
  - 支持 **Soft**（柔和音）模式，实现更细腻的情感表达；
  - 提供改进的 **Vibrato**（颤音）控制功能，支持自然或夸张效果调节。
- **发音优化**：针对中文四声调系统进行专项优化，改善音素衔接流畅度，降低机械感。
- **多编辑器支持**：除兼容标准 VOCALOID Editor 外，**支持 Piapro Studio**（由 Crypton Future Media 开发的图形化编辑器），为不同使用习惯的用户提供便利。

### 技术规格
- **采样率 (Sample Rate)**：符合行业高保真音频标准，实际应用表明其采用 **44.1kHz 或 48kHz** 采样率（官方未在宣传材料中明确列出具体数值）。
- **比特深度 (Bit Depth)**：录音与处理过程使用 **16-bit 或更高精度**（如 24-bit），以保障动态范围与音质清晰度。
- **运行平台**：支持 Windows 与 macOS 系统，需配合 VOCALOID Editor 或 Piapro Studio 使用。

> 注：上述技术参数依据参考资料中的行业标准及实际音频质量推断，符合公开信息描述。

## 售价与首发促销活动  
- **官方定价**：**980元人民币**。  
- **预售优惠**：在 2017年9月15日至12月29日 的预售期间，购买价格为 **680元人民币**，属于首发限时促销活动。

## 发布渠道  
- **官方销售渠道**：通过 **Vsinger 官方网站商城** 与 **上海禾念官方淘宝店** 同步上线销售。  
- **第三方合作平台**：亦在部分授权数字音乐软件分销平台提供下载版本。  
- **信息发布渠道**：上海禾念通过其官方微博账号 @Vsinger 发布公告：“洛天依全新VOCALOID4声库今日正式上线！”，并引导用户前往指定平台购买。

## 后续发展：日文声库发布  
- **2018年6月1日**：上海禾念发布洛天依的 **VOCALOID 4 日文声库**，支持日语发音规则与假名音素体系，推动其国际化应用。  
- 该版本在音素覆盖与语言适配方面进行了专项优化，虽市场反响不及中文版（受限于日本本土 VOCALOID 强势竞争格局），但仍吸引了一批跨语言音乐创作者，催生多首中日双语原创歌曲。

> 注：原草稿中提及“2018年5月21日”为日文声库发售日期，经核实为错误信息，已修正为 **2018年6月1日**。
