# 2015年2月: 参加哔哩哔哩2015拜年祭，演唱歌曲《权御天下》

# 洛天依演唱《权御天下》于哔哩哔哩2015拜年祭

## 事件概述

2015年2月18日，虚拟歌姬洛天依演唱的原创VOCALOID中文歌曲《权御天下》作为《哔哩哔哩2015拜年祭》第一篇章《喜》的压轴节目，通过原创PV动画形式首次公开播出。该节目由哔哩哔哩（B站）出品，在北京时间**2015年2月18日晚8点整**于哔哩哔哩平台全网独播，是当年春节贺岁系列活动的重要组成部分。

## 基本信息

- **时间**：2015年2月18日  
- **地点**：线上（通过哔哩哔哩网站发布）  
- **活动名称**：哔哩哔哩2015拜年祭  
- **参演角色**：洛天依（虚拟歌姬）  
- **演出曲目**：《权御天下》  
- **歌曲类型**：原创VOCALOID中文歌曲  
- **首次公开场合**：哔哩哔哩2015拜年祭 第一篇章《喜》  
- **首次发布时间依据**：原始视频投稿账号“没有龟壳的乌龟”于**2015年2月18日20:00:26**上传完整拜年祭正片（av2075941），《权御天下》在其中首次亮相  

> ✅ 注：部分资料提及“2015年2月14日”为发布日期，但经核查B站原始上传记录及官方节目安排，此说法不准确。正确首播时间为**2015年2月18日**，即农历除夕夜。

## 演出形式与节目结构

- **所属篇章**：第一篇章《喜》  
- **出场顺序**：本篇章最后一个节目（压轴）  
- **表现形式**：原创PV动画 + 战争史诗风格画面 + 动态歌词排版  
- **视觉元素**：包含战鼓、书法、青铜器纹样、古战场沙盘推演等中国风符号  
- **内容主题**：以三国时期司马懿隐忍蓄势、最终通过高平陵之变夺取曹魏大权的历史事件为蓝本，借“权谋”“天下”意象展现力量更迭  
- **节目功能**：从“民俗喜庆”的开篇氛围转向“宏大叙事”的过渡节点，奠定整场拜年祭由“欢乐”到“热血”的情绪基调
