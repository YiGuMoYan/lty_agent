# 2019年: 官宣成为一卡通联名卡合作方

# 2019年：官宣成为一卡通联名卡合作方

## 概述
2019年8月23日，北京市政交通一卡通有限公司（简称“北京一卡通”）与上海禾念信息科技有限公司（Vsinger运营方）联合宣布推出“洛天依联名纪念版交通卡”。此次合作标志着虚拟歌手IP首次与城市公共交通系统实现深度联动，是虚拟偶像在公共服务领域商业化应用的重要突破。

## 合作详情

### 时间
- 官方于 **2019年8月23日** 正式发布合作公告。

### 合作方
- **北京市政交通一卡通有限公司**（发行方）
- **上海禾念信息科技有限公司**（Vsinger运营方，洛天依所属公司）

### 参与角色
- **洛天依**：唯一参与本次联名项目的虚拟歌手。
- 其他Vsinger旗下艺人（如言和、乐正绫等）未参与此批次产品。

### 产品信息
- **产品形式**：实体交通卡（非标尺寸收藏卡），内置符合ISO/IEC 14443 Type A标准的RFID芯片。
- **技术标准**：
  - 支持“交通联合”（China T-Union）标准，可在**全国300余座城市**的公共交通系统中使用。
  - 支持NFC功能，可用于具备NFC写卡能力的智能手机进行模拟刷卡（需设备支持）。
- **设计特点**：
  - 正面印有洛天依剪影与“YI CARD”专属标识，背景为渐变蓝紫色调。
  - 背面标注交通联合标志及独立序列编号。
- **售价**：人民币58元/张（含30元可充值余额）。
- **发行数量**：首批限量发行20,000张，每张独立编号，并附带电子收藏证书。
- **发行范围**：中国大陆地区公开发售。

### 销售模式
- **限购规则**：每人每账号限购2张，需通过实名认证购买。
- **发售渠道**：
  1. **北京一卡通App**（官方主渠道，配额8,000张）
  2. **Vsinger官网商城**（配额待公布）
  3. **Bilibili会员购**（配额6,000张）
  4. **天猫Vsinger旗舰店**（配额待公布）
- **销售表现**：上线后**37分钟内全渠道售罄**。
- **首发销售额**：突破116万元人民币。

### 发布形式
- 本次合作未举办线下发布会。
- 通过**线上联合公告**形式发布，内容同步推送至：
  - 北京一卡通官方微博
  - 北京一卡通微信公众号
  - Vsinger官方B站账号
- 发布内容包括宣传视频与图文通稿，主题聚焦“声动出行”，强调推动城市文化IP建设与绿色出行理念。

## 使用说明
- 卡片可在**北京地铁、公交、有轨电车**以及全国接入“交通联合”系统的公共交通工具上使用。
- 用户可通过**北京一卡通App**进行余额查询、在线充值及NFC写卡操作。
- 卡片不可退换，但已绑定实名账户的卡片支持挂失服务。

## 意义与影响

### 市场反响
- 根据Bilibili用户调研数据，约**72%购买者年龄介于18–30岁**，其中**65%表示因洛天依IP吸引力而首次开通北京一卡通服务**。
- 该卡在二手市场溢价显著，未拆封完整套装在闲鱼、贴吧等平台交易价格一度达到原价**3–5倍**。

### 行业影响
- 被《中国文化报》评为“**2019年度数字文化融合创新典型案例**”。
- 入选《中国文化产业发展报告（2020）》（社会科学文献出版社，P.187），作为虚拟IP赋能传统公共服务的代表性案例。
- 推动后续多个城市交通系统探索与虚拟偶像或文创IP的合作，例如杭州通×初音未来概念提案、深圳通×敦煌文创项目等。

### 历史地位
- 截至2024年，该联名卡被视为中国虚拟偶像跨界合作的标志性事件之一。
- 尽管后续未推出同系列第二弹产品，其成功模式为数字IP进入公共基础设施领域提供了可复制路径。
