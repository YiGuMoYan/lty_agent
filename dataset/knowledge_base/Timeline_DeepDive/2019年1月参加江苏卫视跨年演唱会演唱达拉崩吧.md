# 洛天依参加江苏卫视2019跨年演唱会档案

## 事件概述

2018年12月31日，洛天依作为虚拟歌姬代表参加江苏卫视“2019幸福中国”跨年演唱会，与真人歌手薛之谦同台演唱《达拉崩吧》。此次演出通过实时动作捕捉与全息投影技术实现虚拟形象在现实舞台的呈现，成为当晚节目亮点之一。这是虚拟歌姬首次在中国主流省级卫视跨年舞台上与真人歌手实现现场实时互动合唱，具有里程碑意义，引发公众及媒体对虚拟偶像技术应用的广泛关注。

## 演出详情

- **演出名称**：江苏卫视“2019幸福中国”跨年演唱会  
- **演出时间**：2018年12月31日（实际播出时间为2018年12月31日19:30）  
- **演出地点**：中国澳门金光综艺馆  
- **参演人员**：
  - 薛之谦（真人歌手）
  - 洛天依（虚拟偶像）
- **演唱曲目**：《达拉崩吧》  
- **合作形式**：真人歌手与虚拟偶像现场实时互动合唱  
- **技术实现方式**：实时动作捕捉与全息投影技术
