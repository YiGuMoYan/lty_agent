# 洛天依2018全息演唱会档案

## 事件概要

**BILIBILI MACRO LINK-VISUAL RELEASE × VSINGER LIVE 2018 全息演唱会** 是一场由 Bilibili Macro Link（BML）与 Vsinger 联合主办的虚拟歌手专场全息演唱会。该演出于 **2018年7月20日（星期五）** 在 **上海梅赛德斯-奔驰文化中心** 举行。

本次演唱会是Vsinger系列的重要年度演出，实现了高规格的技术呈现与大规模线上同步直播，成为中文虚拟偶像发展史上的关键节点之一。

- **标志性互动**：在《普通Disco》的演出中，洛天依引导观众根据提示词（如“吉他”、“贝斯”）齐声呼喊VOCALOID成员名字，形成全场联动。此互动模式被粉丝称为“关键词应援”，视频可查证时间戳位于整场演出约第1小时45分钟处。

## 技术评价与客观反馈

- 本次演唱会引进海外顶尖舞美制作团队，采用基于**Realtime Motion Capture（实时动作捕捉）** 的全息影像系统，并结合**现场AR直播技术**，实现虚拟形象与现实舞台的高度融合。
- 演出成功支撑包含20余首歌曲、多个MC环节及嘉宾表演的完整流程，并实现面向440万在线观众的稳定直播。
- 根据当时媒体报道与观众普遍反馈（如“荧光棒的海洋”、“圆满落幕”等描述），演出整体技术呈现稳定，获得广泛正面评价。
- 该场演出被视为推动虚拟偶像技术发展的重要里程碑之一。
