# 2018年5月21日：V4日语声库发布

# 洛天依 V4日语声库发布

- **发布时间**：2018年5月21日  
- **技术平台**：基于 VOCALOID4 引擎开发，需配合兼容的 VOCALOID4 编辑器软件使用；支持 Windows 操作系统，对 CPU、内存和硬盘空间有一定要求  
- **开发与发行方**：由上海禾念信息科技有限公司与雅马哈（Yamaha Corporation）合作开发  
- **声库语言**：日语  
- **声库音色构成**：包含两个独立音色  
  - **Yoshitsune (义経)**：主打清晰、标准的日语发音，适用于主流 J-POP 及流行歌曲演唱  
  - **Sakura (さくら)**：附加特色音色，风格甜美柔和，具有“萌系”或“萝莉”感，提供多样化创作选择  
- **试听曲信息**：  
  - 曲名：《嘘つきは恋のはじまり》（说谎是恋爱的开始）  
  - 创作者：40mP（日本著名 VOCALOID 音乐制作人，本名：清水宏二），以其高质量编曲和对虚拟歌手声音特性的精准把握而闻名  
  - 发布时间：2018年7月8日  
  - 发布平台：niconico动画、YouTube（由40mP本人投稿），Bilibili（由洛天依官方账号同步发布 MV）  
  - 用途：作为洛天依 V4 日语声库的官方宣传曲与核心 DEMO，重点展示 Yoshitsune 音色性能  
- **发售价格**：数字版售价为 9,800 日元（不含税）；实体盒装版价格可能略高  
- **主要销售区域**：面向日本、中国大陆、中国台湾地区、北美、欧洲等全球主要市场  
- **销售渠道**：  
  - 国际渠道：Dreamtonics 官网商店及国际电商平台  
  - 中国大陆渠道：淘宝“上海禾念”官方店铺  
- **语言支持**：安装程序与说明书提供日文、中文（简体/繁体）、英文界面与文档  

> 注：此次 V4 日语双音色声库的推出，标志着洛天依在国际化发展道路上的重要升级，不仅提升了其在日语音乐创作领域的应用能力，也通过高质量的官方试听内容实现了跨区域传播。
