# 普通Disco

# 《普通DISCO》百科档案

## 概述

《**普通DISCO**》是一首由虚拟歌手洛天依“演唱”的VOCALOID中文原创电子舞曲，由音乐人ilem创作并投稿于哔哩哔哩弹幕视频网（Bilibili）。该曲于2015年3月21日19:30正式发布（原视频AV号：AV2175406，BV号：BV1ws411Z7Bf），迅速成为VOCALOID中文圈的现象级作品，是**首个在B站达成“传说”（播放量超100万）与“神话”（播放量超1000万）成就的中文VOCALOID歌曲**，在中国网络音乐发展史上具有里程碑意义。

尽管歌曲由虚拟歌姬演绎，但其影响力突破二次元次元壁，被杨坤、腾格尔、李宇春、汪峰等多位主流艺人正式翻唱或舞台演绎，并登上湖南卫视、浙江卫视、央视等国家级电视平台，实现了从网络亚文化到大众流行文化的跨越。其强烈的132BPM节奏、洗脑旋律及自嘲式歌词催生了海量二创内容，深刻影响中文互联网meme生态，被誉为“二次元国歌”、“V家第一神曲”。

截至2024年6月，该曲在B站原视频累计播放量已突破 **3,800万次**，点赞数超 **250万**，投币数逾 **120万**，收藏数约 **90万**，弹幕数突破 **180万条**，长期位列“B站音乐区经典神曲”榜单，并于2021年获B站“百大视频”荣誉回顾提名，是平台首个播放破千万的VOCALOID中文原创曲之一。

## 创作背景与历史地位

《普通DISCO》由独立音乐人ilem使用VOCALOID技术独立制作，以“普通人也能蹦迪”为核心理念，通过高速节拍（132BPM）、重复性旋律结构与戏谑歌词（如“我怎么这么好看”“来来来来来来来来来”）构建出极具传播力的电子舞曲模板。歌曲发布后迅速引爆B站社区，成为早期中文虚拟歌姬文化的标志性作品。

该曲在B站创造了多项历史性纪录：
- **2016年9月10日**，播放量突破100万，成为VOCALOID中文曲目中首支达成“传说”成就的作品；
- **2020年12月26日**，播放量突破1000万，成为**唯一一首达成“神话”成就的VOCALOID中文歌曲**；
- 截至2024年6月，播放量达 **3,800万+**，点赞 **250万+**，投币 **120万+**，弹幕 **180万+**，为B站最具影响力的原创音乐视频之一。

作为中文VOCALOID运动的奠基之作，《普通DISCO》不仅推动了本土虚拟歌手产业的发展，也成为中国数字音乐“破壁出圈”的典范案例。ilem因此被誉为“中文VOCALOID教父”，该曲亦成为其最具代表性的作品。
